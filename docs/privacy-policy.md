# Privacy Policy for simplesolat

Last updated: November 16, 2025

## Data Collection

simplesolat collects and uses the following data:

### Location Data
- **Purpose**: To automatically detect your prayer zone
- **Usage**: Location is used only to determine which of the 60 Malaysian prayer zones you are in
- **Storage**: Your location is NOT stored or transmitted to any server
- **Control**: You can deny location permission and manually select your zone

### Prayer Times Data
- **Source**: Official JAKIM e-Solat API
- **Storage**: Prayer times are cached locally on your device for offline access
- **Updates**: Data is fetched from JAKIM servers when needed

## Third-Party Services

simplesolat uses:
- JAKIM e-Solat API for prayer times data
- MPT Waktu Solat zone boundaries for GPS-based zone detection

## Data Security

- No personal data is collected
- No user accounts required
- No analytics or tracking
- No advertisements
